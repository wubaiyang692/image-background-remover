# Image Background Remover — MVP 需求文档

**版本：** v0.1  
**日期：** 2026-03-18  
**状态：** 草稿

---

## 1. 产品概述

一个在线图片背景去除工具，用户上传图片后自动去除背景，下载透明 PNG。
无需注册，无需安装，打开即用。

**目标用户：** 电商卖家、设计师、社交媒体运营、普通用户  
**核心价值：** 快速、免费（限额内）、无门槛

---

## 2. 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 纯 HTML + CSS + Vanilla JS（单页） |
| 后端 | Cloudflare Pages Functions（Serverless） |
| 抠图能力 | remove.bg API |
| 部署 | Cloudflare Pages |
| 存储 | 无（图片仅在内存中处理） |

---

## 3. MVP 功能范围

### 3.1 核心功能（必须有）

- **图片上传**
  - 支持点击选择文件
  - 支持拖拽上传
  - 支持格式：JPG、PNG、WebP
  - 文件大小限制：≤ 10MB（remove.bg 免费版限制）

- **背景去除**
  - 上传后自动调用处理，无需手动点击
  - 显示处理进度状态（loading 动画）
  - 处理失败时展示友好错误提示

- **结果展示**
  - 原图 / 结果图左右对比预览
  - 结果图背景用棋盘格纹显示透明区域

- **下载**
  - 一键下载透明背景 PNG
  - 文件名：`removed-bg-{原文件名}.png`

### 3.2 不在 MVP 范围（后续迭代）

- 用户账号 / 登录
- 历史记录
- 批量处理
- 背景替换（换颜色/换图片）
- 付费订阅
- 多语言

## 5. API 设计

### POST `/api/remove-bg`

**请求**
```
Content-Type: multipart/form-data
Body: { image: File }
```

**响应（成功）**
```
Content-Type: image/png
Body: <透明背景图片二进制>
```

**响应（失败）**
```json
{ "error": "错误描述" }
```

**后端逻辑**
1. 接收前端上传的图片（内存，不落盘）
2. 转发给 remove.bg API（`https://api.remove.bg/v1.0/removebg`）
3. API Key 从 Cloudflare 环境变量 `REMOVE_BG_API_KEY` 读取
4. 将结果直接流式返回给前端

## 6. 非功能需求

| 项目 | 要求 |
|------|------|
| 响应时间 | 处理完成 ≤ 10s（取决于 remove.bg） |
| 隐私 | 图片不落盘、不存储、不记录 |
| 兼容性 | Chrome / Safari / Firefox 最新版 |
| 移动端 | 响应式，支持手机上传 |
| HTTPS | Cloudflare 自动提供 |

---

## 7. 部署流程

1. 代码推送到 GitHub 仓库
2. Cloudflare Pages 连接该仓库，自动构建
3. 在 Cloudflare Pages → Settings → Environment Variables 中添加：
   - `REMOVE_BG_API_KEY` = `你的 remove.bg API Key`
4. 绑定自定义域名（可选）
